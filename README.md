# TextFileCLI
CLI program written in java for performing file operations.

## Dependencies
Install these dependencies on Ubuntu/Debian
```
sudo apt install openjdk-17-jdk openjdk-17-jre
```

## Program Usage
### Create a text file
```
java src/Main.java create test.txt
```

### Read a text file 
```
java src/Main.java read test.txt
```

### Write to a text file 
```
java src/Main.java write test.txt test
```
